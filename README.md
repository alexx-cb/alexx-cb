Me apasiona crear soluciones eficientes, escalables y bien estructuradas, con especial interés en la calidad del código y la optimización del rendimiento. Actualmente continúo ampliando mis conocimientos en el ecosistema Python, explorando Flask, Django y el análisis de datos con Pandas y NumPy.
